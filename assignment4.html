Quick Description:

For assignment 4, I created a very basic 2D Action/Platformer/Sidescroller game using Unity. The sprite assets that I used can be found here and here. The main objective of the game is to survive as long as possible by not dying to the enemy red ovals. The way the player achieves this is by using the few mechanics given to the player. Basic movement mechanics allow the player to maneuveur left (using 'A'), right (using 'D'), and up (using 'Space'). The player can also utilize gravity to go down.

The main mechanic that allows the player to survive is the ability to fire projectiles. If the player's projectile (orange) collides with an enemy red oval, the oval is destroyed. If the player collides with an enemy projectile (green), then the player character loses some hp (total HP is 100, 1 projectile DMG = 1). Once the player loses all HP, the player character is destroyed. There are also health boxes that spawn around the map allowing for the player to regain HP, all the player has to do is collide with the box. The enemy red ovals also use the same spawning mechanic as the health boxes, and have a couple different spawn points around the map. If the player is to die, or the game has a problem, they can use the reset button ('L') to reset the game.
